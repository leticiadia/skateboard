## 🛹 Skateboard

Desenvolvi esse projeto como parte do meu portfólio, unindo design moderno, performance e boas práticas de desenvolvimento front-end.

O projeto Skateboard reflete minha paixão pelo skate e pela criação de interfaces que proporcionam uma experiência fluida e visualmente marcante.

### 🖼️ Preview do Projeto

![skateboard-preview](/src/assets/images/skateboard-preview.png)

### 🚀 Tecnologias Utilizadas

O projeto foi construído com foco em componentização e responsividade, utilizando:

- [ReactJS](https://react.dev/)
- [Vite.js](https://vite.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [TailwindCSS](https://tailwindcss.com/)

### ⚙️ Como Executar o Projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/skateboard.git

# Acesse a pasta do projeto
cd skateboard

# Instale as dependências
npm install

# Execute o projeto em ambiente de desenvolvimento
npm run dev

```
